§4 · RESOURCE SCALING

    
Solution space grows factorially. Encoding grows polynomially.

    

      For n nodes on a complete graph there are
      (n−1)!/2 distinct Hamiltonian cycles. The QUBO
      encoding in backend/vrp.py uses
      n(n−1) binary variables — one qubit per directed
      edge. The gap between these curves is the argument for quantum: the search
      space explodes faster than the resources required to query it.
    

    

      
    

    

      |routes| = (n−1)! / 2 · Hamiltonian cycles on Kn

      qubits = n(n−1) · exact count from backend/vrp.py
